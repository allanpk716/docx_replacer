# S02 UAT: 应用图标

## 前置条件
- DocuFiller 已构建（`dotnet build -c Release`）
- Windows 文件资源管理器可用

## 测试用例

### TC-01: exe 文件图标
1. 在文件资源管理器中导航到 `bin/Release/net8.0/win-x64/` 目录
2. 找到 `DocuFiller.exe`
3. **预期**：exe 文件在文件资源管理器中显示蓝色文档+绿色对勾图标，而非默认的通用 exe 图标
4. 右键 → 属性 → 查看图标
5. **预期**：属性对话框中显示 DocuFiller 专属图标

### TC-02: 任务栏图标（所有窗口）
1. 运行 DocuFiller.exe
2. **预期**：任务栏显示 DocuFiller 专属图标（蓝色文档+绿色对勾）
3. 点击"文档填充"启动填充流程
4. 如触发下载进度窗口，**预期**：下载进度窗口的任务栏图标为 DocuFiller 图标
5. 打开"更新设置"窗口
6. **预期**：更新设置窗口的任务栏图标为 DocuFiller 图标
7. 打开"审核清理"窗口
8. **预期**：清理窗口的任务栏图标为 DocuFiller 图标

### TC-03: MainWindow 自定义标题栏图标
1. 运行 DocuFiller.exe
2. 观察主窗口顶部自定义标题栏左侧
3. **预期**：显示 DocuFiller 图标（16x16），不是 📄 emoji
4. 图标应清晰锐利（使用 HighQuality 缩放模式）
5. 图标右侧紧邻应用标题文本

### TC-04: 标准窗口标题栏图标
1. 打开"更新设置"窗口
2. **预期**：标准 Windows 标题栏左侧显示 DocuFiller 图标
3. 打开"审核清理"窗口
4. **预期**：标准 Windows 标题栏左侧显示 DocuFiller 图标
5. 如有下载进度窗口，**预期**：标题栏左侧显示图标

### TC-05: Alt+Tab 切换器图标
1. 运行 DocuFiller.exe
2. 按 Alt+Tab
3. **预期**：应用切换器中 DocuFiller 显示专属图标，不是空白或通用图标

## 边界情况
- **高 DPI 显示器**：图标在不同缩放比例（100%/150%/200%）下应清晰
- **多显示器**：图标在不同 DPI 的显示器间移动时应正确显示
- **资源管理器大图标视图**：在"大图标"视图下 exe 图标应正确渲染
